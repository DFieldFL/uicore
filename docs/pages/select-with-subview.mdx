# Select With Subview--- docs/pages/select-with-subview.mdx


Select with Subview renders an alternate view when the user selects a specific option in the dropdown. The alternate view is rendered within the drop down menu.
The component is built on top of uikit's standard select component.

## Import

```js
import { SelectWithSubview, EnvironmentTypeSubForm } from '@wings-software/uikit'
```

## Stack Trace Examples

```js live=true
<SelectWithSubview items={[{value: "1234_envId", label: "Env1"},{value: "5678_envId", label: "Env2"}]} changeViewButtonLabel="+ Environment" subview={<EnvironmentTypeSubForm onSubmit={(values)=>console.log(values)}/>} />
```
