# Vertical

`<Layout.Vertical/>` lays out children in a vertically spaced layout.

## Props

- spacing: number

## Playground

```js live=true
<Layout.Vertical spacing={25}>
  <div style={{ height: '100px', background: 'red' }}></div>
  <div style={{ height: '100px', background: 'green' }}></div>
  <div style={{ height: '100px', background: 'blue' }}></div>
</Layout.Vertical>
```
