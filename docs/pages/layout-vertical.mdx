# Vertical --- docs/pages/layout-vertical.mdx

`<Layout.Vertical/>` lays out children in a vertically spaced layout.

```js live=true
<Layout.Vertical spacing="large">
  <div style={{ height: '100px', background: 'var(--grey-300)' }}></div>
  <div style={{ height: '100px', background: 'var(--green-500)' }}></div>
  <div style={{ height: '100px', background: 'var(--blue-500)' }}></div>
</Layout.Vertical>
```
