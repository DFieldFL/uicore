# Card --- docs/pages/card.mdx

Card component reuses Blueprint's card and implements the look and feel of [CD Next Gen](https://harness.atlassian.net/wiki/spaces/UI/pages/668271019/Card)
## Import

```js
import { Card, CardBody } from '@wings-software/uikit'
```


## Types

```js
interface CardProps {
  elevation: number (0|1|2|3|4);
  interactive: boolean (false);
  selected: boolean (false);
  onClick?: (e: React.MouseEvent<HTMLDivElement>) => void;
}

// Example
<Card interactive="true">
    <Text style={{marginTop: '5px'}} font="medium">Jenkin Artifact</Text>
</Card>

```
