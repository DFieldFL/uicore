# Typography

## Font Family

[Nunito](https://fonts.google.com/specimen/Nunito?selection.family=Nunito) is Harness default text font. This font is enforced to whole document so when you build your component, you don't need to specify it.

## Font Sizes

- xsmall
- small
- default (no size is needed)
- medium
- large
- xlarge
- xxlarge
- xxxlarge
- huge

## Font Weight

- bold
