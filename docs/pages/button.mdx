# Button --- docs/pages/button.mdx

Button component reuses Blueprint's Button and implement the look and feel of [HDL](https://projects.invisionapp.com/d/?origin=v7#/projects/prototypes/17628604).

## Import

```js
import { Button } from '@wings-software/uikit'
```

## Primary Button

```js live=true
<Layout.Horizontal spacing="small">
  <Button intent="primary" text="Text Button" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" icon="plus" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" text="Text & Icon" icon="plus" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" text="Right Icon" rightIcon="chevron-right" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" text="Left Icon" icon="chevron-left" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" text="Down Icon" rightIcon="chevron-down" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" text="Disabled" disabled icon="cog" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" minimal text="Text Button" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" minimal icon="plus" onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" minimal text="Text & Icon" icon="plus" onClick={() => alert('Hello World')}></Button>
  <Button
    intent="primary"
    minimal
    text="Right Icon"
    rightIcon="chevron-right"
    onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" minimal text="Left Icon" icon="chevron-left" onClick={() => alert('Hello World')}></Button>
  <Button
    intent="primary"
    minimal
    text="Down Icon"
    rightIcon="chevron-down"
    onClick={() => alert('Hello World')}></Button>
  <Button intent="primary" minimal text="Disabled" disabled icon="cog" onClick={() => alert('Hello World')}></Button>
</Layout.Horizontal>
```
