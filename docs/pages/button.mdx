# Button

Button component reuses Blueprint's Button and implement the look and feel of [HDL](https://projects.invisionapp.com/d/?origin=v7#/projects/prototypes/17628604).

## Import

```js
import { Button } from 'ui-platform'
```

## Examples

```js live=true
<Button text="Let go" icon="refresh" onClick={() => alert('Hello World')}></Button>
```
