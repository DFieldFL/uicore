# Switch --- docs/pages/switch.mdx

Switch component reuses Blueprint's Switch and implements the look and feel of [HDL](https://projects.invisionapp.com/d/?origin=v7#/projects/prototypes/17628604).
Switch API is exactly the same as [Blueprint's Switch](https://blueprintjs.com/docs/#core/components/switch).

## Import

```js
import { Switch } from '@wings-software/uikit'
```

## Switches

```js live=true
<Switch label="Not Selected" />
<Switch label="Selected" checked/>
<Switch label="Disabled" disabled/>
<Switch label="Disabled and Selected" disabled checked />
```
