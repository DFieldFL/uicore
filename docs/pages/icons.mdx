# Icons --- docs/pages/icons.mdx

All icons in UIKit are in svg. You can use the icons as a separate React component. [Button](/button) supports them as well.

To use a Harness icon, first import `Icons` from UIKit.

```js
import { Icons } from '@wings-software/uikit'
```

Then render the icons as a React component. For example:

```js live=true
<Icons.ServiceAws height="64px" width="64px" />
```

`Icons` supports standard [HTML attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes). To keep it simple, mostly you should use only `height`, or `width`.

## Harness Icons

```js render=true
<div
  style={{
    padding: '20px',
    display: 'grid',
    background: 'var(--grey-100)',
    gridGap: '20px',
    gridTemplateColumns: 'repeat(auto-fit, minmax(50px, 1fr))',
    borderRadius: '5px'
  }}>
  {Object.entries(Icons).map(([name, Icon]) => {
    {
      /* Blueprint does not work in NextJS with usePortal=true */
    }
    return name.includes('HarnessLogo') ? null : (
      <Tooltip key={name} content={name + ' (click to copy)'} usePortal={false}>
        <button
          onClick={() => Utils.copy(`<Icons.${name} height="64px" width="64px" />`)}
          style={{
            all: 'unset',
            cursor: 'pointer'
          }}>
          <Icon height="36px" width="36px" />
        </button>
      </Tooltip>
    )
  })}
</div>
```

### Harness Logos

There are currently two Harness logo icons.

```js render=true
<div
  style={{
    padding: '20px',
    display: 'grid',
    background: 'var(--grey-100)',
    gridGap: '20px',
    gridTemplateColumns: 'repeat(auto-fit, minmax(250px, 1fr))',
    borderRadius: '5px'
  }}>
  {Object.entries(Icons)
    .filter(([name]) => name.includes('HarnessLogo'))
    .map(([name, Icon]) => {
      return (
        <Tooltip key={name} content={name + ' (click to copy)'} usePortal={false}>
          <button
            onClick={() => Utils.copy(`<Icons.${name} height="64px" width="64px" />`)}
            style={{
              all: 'unset',
              cursor: 'pointer'
            }}>
            <Icon key={name} height="48px" />
          </button>
        </Tooltip>
      )
    })}
</div>
```

## How to add an icon?

To add an icon into UIKit, copy the icon into UIKit's `./src/icons` folder then run

```
yarn ui:icons
```
