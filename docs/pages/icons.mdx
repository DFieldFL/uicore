# Icon --- docs/pages/icon.mdx

import { Button } from '../static'

All icons in UIKit are in svg format. You can use the icons as a separate React component or using them in [Button](/button).

Note that `<Icon/>` component supports multiple colors in svg file, but when combining with `<Button/>` component, fill and stoke colors in svg files are reset to the text color of the button.

To use a Harness icon, first import `Icon` from UIKit.

```js
import { Icon } from '@wings-software/uikit'
```

Then render icon as a React component.

```js live=true
<Layout.Horizontal spacing="small">
  <Icon name="service-gotlab" size={64} />
  <Icon name="service-datadog" size={64} />
  <Icon name="service-github" size={64} />
  <Icon name="service-gcp" size={64} />
  <Icon name="service-jenkins" size={64} />
</Layout.Horizontal>
```

You can pass styled props to Icon as well.

```js live=true
<Icon name="heart" size={64} color="blue500" padding="small" border={{ color: 'blue400' }} />
```

## Harness Icons

```js render=true
<div
  style={{
    padding: '20px',
    display: 'grid',
    background: 'var(--grey-100)',
    gridGap: '20px',
    gridTemplateColumns: 'repeat(auto-fit, minmax(50px, 1fr))',
    borderRadius: '5px'
  }}>
  {Object.keys(HarnessIcons).map(name => {
    {
      /* Blueprint does not work in NextJS with usePortal=true */
    }
    return name.includes('HarnessLogo') ? null : (
      <Tooltip key={name} content={name + ' (click to copy)'} usePortal={false}>
        <button
          onClick={() => Utils.copy(name)}
          style={{
            background: 'none',
            border: 'none'
          }}>
          <Icon name={name} size={32} padding="small" />
        </button>
      </Tooltip>
    )
  })}
</div>
```

## How to add an icon?

To add an icon into UIKit, copy the icon into UIKit's `./src/icons` folder then run

```
yarn ui:icons
```
