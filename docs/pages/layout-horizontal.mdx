# Horizontal

`<Layout.Horizontal/>` lays out children in a horiztonally spaced layout.

## Props

- spacing: number

## Playground

```js live=true
<Layout.Horizontal spacing={25}>
  <div style={{ height: '100px', width: '200px', background: 'red' }}></div>
  <div style={{ height: '100px', width: '300px', background: 'green' }}></div>
  <div style={{ height: '100px', background: 'blue', flexGrow: 1 }}></div>
</Layout.Horizontal>
```
