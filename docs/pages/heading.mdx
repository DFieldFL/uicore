# Heading

# Heading component

I'd like to add some description here!!!

## Props

- `level?`: 1-> 6
- `inline?`: `boolean`: If `true`, render the component as inline block. Default is `false`.
- `intent?`: [Intent](/intent).

## Samples

### Render as block

```js live=true
<Heading level={1}>Heading level 1</Heading>
<Heading level={2}>Heading level 2</Heading>
<Heading level={3}>Heading level 3</Heading>
<Heading level={4}>Heading level 4</Heading>
<Heading level={5}>Heading level 5</Heading>
<Heading level={6}>Heading level 6</Heading>
```

### Render as inline

```js live=true
<Heading level={1} inline>Heading level 1</Heading>
<Heading level={2} inline>Heading level 2</Heading>
<Heading level={3} inline>Heading level 3</Heading>
<Heading level={4} inline>Heading level 4</Heading>
<Heading level={5} inline>Heading level 5</Heading>
<Heading level={6} inline>Heading level 6</Heading>
```

### Intent

Heading supports [Intent](/intent) where an intent color is applied.

```js live=true
<Heading intent="primary">Heading with primary intent</Heading>
<Heading intent="success">Heading with success intent</Heading>
<Heading intent="warning">Heading with warning intent</Heading>
<Heading intent="danger">Heading with danger intent</Heading>
```
