# TextInput --- docs/pages/link.mdx

## Import

```js
import { TextInput } from '@wings-software/uikit'
```

## Idle state

```js live=true
<TextInput label="Label" defaultValue="" placeholder="Placeholder text" />
```

## Error state

```js live=true
<TextInput
  label="Label"
  defaultValue="Now in error state"
  intent="danger"
  errorText="Not a valid data. Please try again."
/>
```

## Error state (in popover)

```js live=true
<TextInput
  label="Label"
  defaultValue="Show Error icon"
  errorText="Not a valid data. Please try again."
  intent="danger"
  errorInPopover
/>
```

## Success state

```js live=true
<TextInput label="Label" placeholder="Placeholder text" intent="success" />
```

## Disabled state

```js live=true
<TextInput label="Label" placeholder="Placeholder text" disabled />
```

## Icons

```js live=true
<TextInput
  label="Label"
  placeholder="Search"
```
