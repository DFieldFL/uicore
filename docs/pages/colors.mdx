# Colors --- docs/pages/colors.mdx

All colors are defined as global CSS variables. That means you can use the colors anywhere in your CSS and React inline styles without having to import.

As a matter of rule, no hex code should be used in any codebase. All colors must be defined in `colors.css` before they can be referred to.

To use a Color Reference, you'll need to import `Color`.

It's recommended to use colors in the NextGen UI Color Palette and stop using colors in the Legacy Color Palette.

```js
import { Color } from '@wings-software/uikit'
```

## Outline Colors

Outline Colors are available only under NextGen UI (not available under wingsui).

```js live=true
<Layout.Horizontal spacing="medium">
  <Container width={100} height={100} border={{ color: 'dark' }}></Container>
  <Container width={100} height={100} border={{ color: 'light' }}></Container>
  <Container width={100} height={100} border={{ color: 'red' }}></Container>
```
