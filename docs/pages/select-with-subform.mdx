# Select With Secondary View --- docs/pages/select-with-secondary-view.mdx


Select with Secondary View renders an alternate view when the user selects a specific option in the dropdown. The alternate view is rendered within the drop down menu.

## Import

```js
import { SelectWithSecondaryView, EnvironmentTypeSubForm } from '@wings-software/uikit'
```

## Stack Trace Examples

```js live=true
<SelectWithSecondaryView items={[{value: "1234_envId", label: "Env1"},{value: "5678_envId", label: "Env2"}]} changeViewButtonLabel="+ Environment" subForm={<EnvironmentTypeSubForm onSubmit={(values)=>console.log(values)}/>} />
```
