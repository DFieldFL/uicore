# Column Chart --- docs/pages/ColumnChart.mdx

## Import

```js
import { ColumnChart } from '@wings-software/uikit'
```

## Column Chart with default values, custom height and custom width

```js live=true

<Layout.Horizontal spacing="small">
  <ColumnChart height={300} width={400} >
  </ColumnChart>
</Layout.Horizontal>

```

## Column Chart with custom options

```js live=true

<Layout.Horizontal spacing="small">
  <ColumnChart options={{ chart: { type: 'column'}, series:[ { name:'Deployments' , data:[1,2,3,4]} ], credits: {enabled: false} }} >
  </ColumnChart>
</Layout.Horizontal>
